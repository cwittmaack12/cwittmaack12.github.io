Certification

DevMountain

In September 2022 I began my journey with DevMountain. The bootcamp was a total of 16 weeks with 40+ hours per week allocated towards this endeavor. The first eight weeks, referred to as "Foundations" primarily focused on full-stack web development. The tech stack was prominently based in Javascript, HTML, & CSS. The second eight weeks focused heavily on Data Analysis including writing SQL queries, cleaning & transforming data in Python using Jupyter Notebook, doing an Exploratory Data Analysis, building machine learning models, creating data visualizations, & presenting findings.

The first and second parts of the program both culminated with a capstone project. In the first capstone I created a full stack web app to plan your day & in the second project I analyzed time series data to try and determine what the trend in hurricane activity looks like.
